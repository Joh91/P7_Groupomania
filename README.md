# P7_Groupomania

## Installations
> Clonez le repot 
> Allez à la racine du projet depuis votre terminal

> Backend 
- Installer nodejs et npm 
- cd backend 
- npm install
- nodemon serve

> Frontend 
Le projet à été réalisé sous Vuejs 
- cd frontend / groupomania-frontend 
- npm install 
- npm run serve 

> Url 
- http://localhost:8080/
